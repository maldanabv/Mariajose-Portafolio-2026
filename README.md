# MJ Digital Studio

Portfolio personal de María José Aldana Bravo, creadora digital especializada en diseño visual, branding y dirección creativa.

## Descripción

Este proyecto es un sitio web estático que presenta el portfolio de MJ Digital Studio. Incluye secciones como hero, sobre mí, servicios, habilidades, testimonios y contacto.

## Cambios Realizados

### Optimizaciones en HTML
- **Texto de descripción acortado**: En la sección hero, el texto "Creo experiencias digitales a través del diseño visual, el branding y la dirección creativa. Te doy un recorrido por mis habilidades." fue simplificado a "Diseño experiencias digitales con branding y dirección creativa." para mayor concisión.
- **Código HTML compactado**: La sección `hero-image` fue comprimida para reducir el tamaño del archivo HTML.

### Interactividad y Estilos
- **Botones interactivos**: Agregados efectos hover a todos los botones (`.btn-talk`, `.btn-primary`, `.btn-secondary`, `.contact-form button`) con:
  - Cambio de color de fondo a `hwb(353 18% 55%)`
  - Sombra (`box-shadow`) y leve elevación (`transform: translateY(-2px)`)
  - Transiciones suaves (`transition: 0.3s ease`)
- **Enlaces de navegación**: Los enlaces en la barra de navegación ahora cambian a color `hwb(353 18% 55%)` en hover con transición suave.
- **Hipervínculo de LinkedIn**: El texto "LinkedIn" en el footer ahora es un enlace funcional a `https://www.linkedin.com/in/maria-jose-aldana-bravo-02a86033b/`, con color personalizado `#fde2e2`.
